# moqui-training


## Overview
This repository contains my Moqui Framework training setup with MySQL.

## Database Configuration
- File: runtime/conf/MoquiDevConf.xml
- MySQL host: 127.0.0.1
- Database: moqui
- User: root

## Components / Data
- Custom components are in component/moqui/


## How to Run
1. Navigate to moqui-framework folder
2. Run:
   ./gradlew getRuntime
   ./gradlew load
